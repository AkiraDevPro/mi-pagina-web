Control de Secuencia de Imágenes

Fase Inicial u Origen

Carga inicial de recursos multimedia configurados localmente desde el directorio del proyecto.

Fase de Transición

El motor de Bootstrap ejecuta el desplazamiento lateral controlado por eventos asíncronos nativos.

Fase de Bucle

Al alcanzar el último nodo de la lista, el sistema reinicia la secuencia de forma continua.

Estado del Entorno: Ejecución en servidor de desarrollo local. Las dependencias funcionales CSS y JS se cargan de forma nativa desde rutas relativas para garantizar la operatividad offline.
